# LocalTransfer

> 基于 Node.js 的简单局域网内文件共享服务

## 一、运行环境

- Node.js v20.0+

- YARN

## 二、安装依赖

进入项目目录后，执行 Shell 命令：

```shell
yarn
```

## 三、运行步骤

1. 在项目目录中，执行 Shell 命令：

    ```shell
    yarn build
    yarn start
    ```

2. 在交互命令行中：

    - 输入 `$` 并按回车刷新接收服务器列表

    - 输入文件路径并按回车上传目标文件到所有接收服务器



## 四、同传前提

1. 两台或两台以上希望共享文件的主机

2. 处于同一局域网内（请注意是否配置了 VLAN 等）

3. 希望共享文件的主机都启动服务
